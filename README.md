# Rusty Design Patterns

I'm following along Gang of Four and figured the best way to learn the patterns is to apply them to self-contained examples. Each directory under `src` has a README with the pattern and notes from the book, and a reference implementation and some tests.

## Skipped Patterns

I'll skip some patterns that have a clear built-in implementation in Rust and enumerate them below.
